# Stock Portfolio Tracker – Python Internship Project

## Project Description

The **Stock Portfolio Tracker** is a beginner-friendly Python application developed as part of my **Python Internship**. The objective of this project was to strengthen my understanding of core Python programming concepts by building a practical, menu-driven application.

This application allows users to maintain a simple virtual stock portfolio by adding stock details, viewing their holdings, updating information, calculating the total portfolio value, and managing their investments through an easy-to-use console interface. The project focuses on implementing programming logic rather than integrating real-time stock market data.

Throughout the development process, I applied fundamental Python concepts such as variables, data types, dictionaries, functions, loops, conditional statements, user input handling, and basic data manipulation. The project follows a modular and structured approach, making the code easy to read, understand, and maintain.

### Key Features

* Add stocks to the portfolio.
* View all stocks stored in the portfolio.
* Update stock information when required.
* Remove stocks from the portfolio.
* Calculate the total portfolio value.
* Menu-driven interface for smooth user interaction.
* Simple and beginner-friendly console application.

### Technologies Used

* Python 3

### Python Concepts Implemented

* Variables and Data Types
* Dictionaries
* Functions
* Loops
* Conditional Statements (`if`, `elif`, `else`)
* User Input and Output
* String Formatting
* Basic Error Handling

### Learning Outcomes

By completing this project, I gained hands-on experience in organizing code into functions, managing data using dictionaries, implementing menu-driven programs, and solving real-world problems using Python. This project also improved my logical thinking, debugging skills, and understanding of Python fundamentals, providing a strong foundation for building more advanced applications in the future.

This project represents my practical learning during the internship and demonstrates my ability to apply core Python concepts to create a functional console-based application.
